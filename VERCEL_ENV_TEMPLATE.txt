##############################################################################
# TROUBLESHOOTING
##############################################################################

If frontend can't connect to backend:
‚òê Verify REACT_APP_API_URL is set correctly in Vercel
‚òê Redeploy frontend after adding environment variable
‚òê Check browser console for actual error
‚òê Test backend URL directly in browser: /api/health
‚òê Ensure backend CORS allows your Vercel domain

If you see CORS errors:
‚òê Update CORS_ORIGINS in Render with your Vercel URL
‚òê No trailing slashes in any URLs
‚òê Render automatically redeploys after env var change
‚òê Clear browser cache and reload
